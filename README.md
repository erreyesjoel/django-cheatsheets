Colección de **apuntes y chuletas de Joel Erreyes** sobre el ecosistema Django: desarrollo tradicional con **templates y HTML**, uso de **JavaScript** integrado, manejo del **ORM**, y creación de **APIs con Django REST Framework (DRF)**.

> [!IMPORTANT]
> Para la creación y gestión de APIs se utiliza **DRF**; *Django Ninja no está incluido porque no lo he trabajado todavía*.

El objetivo es disponer de una referencia rápida, clara y práctica sin tener que navegar por documentación extensa cada vez.

---

## 📚 Contenido

Este repositorio recopila notas breves y ejemplos sobre:

* **Django tradicional**: vistas, URLs, templates, formularios, autenticación, middleware.
* **Django REST Framework (DRF)**: serializers, viewsets, routers, permisos, autenticación, testing.
* **ORM de Django**: consultas, relaciones, anotaciones, optimización.
* **Frontend básico**: HTML, templates, JavaScript integrado con Django.
* **Snippets reutilizables**: fragmentos de código listos para copiar y pegar.
* **Buenas prácticas**: estructura de proyectos, patrones, seguridad y rendimiento.

---

## 🎯 Filosofía del repositorio

* **Rápido de consultar**: ejemplos cortos y directos.
* **Práctico**: orientado a resolver dudas comunes del día a día.
* **Personal**: basado en la experiencia y estilo de trabajo de **Joel Erreyes**.
* **Vivo**: pensado para crecer y actualizarse con el tiempo.
